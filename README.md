# MQ
仿 RabbitMQ 实现消息队列

## 1. 项目介绍
RabbitMQ 是一个非常知名、功能强大且广泛使用的消息队列。该项目仿照 RabbitMQ 模拟实现一个简单的消息队列。

## 2. 开发环境
- Linux (Ubuntu-24.04)
- VSCode
- g++/gdb
- Makefile

## 3. 技术选型
- **开发主语言**：C++
- **序列化框架**：Protobuf 二进制序列化
- **网络通信**：
  - 自定义应用层协议 + muduo 库
  - 对 TCP 长连接的封装，并且使用 epoll 的事件驱动模式，实现高并发服务器与客户端
- **源数据信息数据库**：SQLite3
- **单元测试框架**：Gtest
